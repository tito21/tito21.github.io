YASSG (yet another static site generator)

Usage:

  1. Write a post in Markdown under the _post directory and include all assets in _assets.
  2. Run python yassg.py.
  3. cd into public and commit&push all changes
  4. cd back into the root directory and commit&push

The public folder is a manage as a git submodule as the instructions on the hugo docs. Do not delate the folder or any of its contents

The steps 2 and 3 are know manage by deploy.sh. Only writing post and committing to the root repo should be done manually.

Post with a field draft in the frontmatter will be ignored when building the site.

# This is just to test some stuff

import pathlib

site_root = 'https://site.com'

with open(path_site/'robots.txt', 'w') as f:
    f.write("Sitemap: {}/sitemap.xml\n".format(site_root))
